# 📱 iOS 设备信任开发者证书详细指南

## 问题说明
当你第一次在真机上运行使用开发者证书签名的应用时，iOS 会显示：
```
"无法启动应用，因为开发者 App 证书不受信任"
```

这是 iOS 的安全机制，需要手动信任开发者证书。

---

## 🔧 解决步骤（请在你的 iPhone 上操作）

### 第 1 步：打开"设置"应用
📍 在 iPhone 主屏幕找到灰色齿轮图标的"设置"
```
[主屏幕]
┌─────────────────┐
│  ⚙️ 设置        │ ← 点击这里
│  📱 电话        │
│  📧 邮件        │
└─────────────────┘
```

---

### 第 2 步：进入"通用"
📍 在设置列表中向下滚动，找到"通用"（通常在第三或第四个位置）

### 第 3 步：找到"VPN 与设备管理"
📍 在"通用"页面中，向下滚动找到这个选项


**如果找不到？**

## ❓ 常见问题

### Q1: 找不到"VPN 与设备管理"选项？
**原因：** 设备上还没有安装开发者应用

**解决方法：**
1. 在 Xcode 中，确保选择了你的真机设备（不是模拟器）
2. 点击运行按钮 ▶️
3. 等待应用安装到设备上（会显示安装进度）
4. 安装完成后，再次检查"设置 → 通用 → VPN 与设备管理"

### Q2: 点击"信任"后还是打不开应用？
**解决方法：**
1. 完全关闭应用（双击 Home 键或上滑手势，关闭应用卡片）
2. 重启 iPhone
3. 再次尝试打开应用

### Q3: 看到多个开发者证书，应该信任哪个？
**解决方法：**
- 信任与你的 Apple ID 或组织名称匹配的证书
- 在 Xcode 中查看：项目设置 → Signing & Capabilities → Team
- 信任显示相同名称的证书

### Q4: 信任后过一段时间又不能用了？
**原因：** 免费开发者证书有效期只有 7 天

**解决方法：**
1. 在 Xcode 中重新运行应用（自动续签）
2. 或者：加入 Apple Developer Program（99 美元/年）获得 1 年有效期证书

## 📝 快速参考

完整路径：
```
设置
  → 通用
    → VPN 与设备管理
      → 开发者 App
        → [你的开发者账号]
          → 信任
```

---

## 💡 提示
- 这个操作**只需要做一次**（除非证书过期或重新安装）
- 信任证书是安全的，因为是你自己的开发者账号
- 如果要撤销信任，在同一页面点击"删除 App"即可
