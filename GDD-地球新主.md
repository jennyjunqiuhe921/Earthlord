## 1. 游戏概述

### 1.1 游戏愿景
《地球新主》是一款"希望朋克"风格的LBS生存游戏，玩家通过现实世界的行走，在末日废墟上圈占领地、探索资源、建造家园，与其他幸存者协作，重建人类文明。

**核心体验**:
- **探索的刺激**: 每一次外出都可能发现新的资源点
- **占有的满足**: 用脚步丈量的土地真正属于你
- **建造的成就**: 从无到有打造自己的避难所
- **社交的温暖**: 在末日废土中找到志同道合的伙伴
- **生存的挑战**: 饱食度和水分的持续压力

### 1.2 玩家目标
**短期目标** (1-7天):
- 圈下第一块领地
- 完成第一次探索
- 建造第一个篝火
- 加入一个频道

**中期目标** (1-4周):
- 建立完整的生存基地（庇护所+仓库+生产设施）
- 探索周边5公里所有POI
- 完成10次交易
- 登上区域排行榜

**长期目标** (1-3个月):
- 拥有多块领地形成领地网络
- 收集所有稀有建筑
- 成为区域领袖（排行榜Top 10）
- 解锁所有传说成就

### 1.3 核心循环
```
外出行走 → GPS圈地/探索POI → 获得资源 → 建造设施/交易物品 → 提升实力 → 探索更远
```

## 2. 世界观与叙事

### 2.1 背景故事

**时间线**:
- **2048年**: "终末战争"爆发
  - 持续72小时的全球核战争
  - 99%人口在战争和随后的瘟疫中消失
  - 所有政府瓦解，旧秩序崩塌

- **2048-2051年**: "漫长的寒冬"
  - 核冬天导致气候剧变
  - 幸存者在地下避难所艰难求生
  - 自然开始缓慢恢复

- **2051年3月**: 《新开拓者协议》颁布
  - 最后的卫星网络重启
  - 全球幸存者委员会成立
  - 新的土地法则确立

- **2051年春天**: 游戏开始时刻
  - 玩家走出避难所
  - 开始重建之旅

### 2.2 世界设定

**《新开拓者协议》核心条款**:
1. **圈地法则**: "凡以双足丈量之无主土地，皆为开拓者所有"
2. **活跃原则**: 30天内必须开始建设，90天内必须保持活跃
3. **交易自由**: 鼓励以物易物，禁止暴力掠夺
4. **互助精神**: 协作共赢者将获得更多权益
5. **废墟搜刮权**: 所有旧世界遗迹可自由搜刮

**世界现状**:
- 城市变成废墟，但建筑结构大多完好
- 植物顽强生长，废土上出现绿色生机
- 动物回归城市，生态正在重建
- 幸存者散落各地，通过无线电联络
- 资源有限，需要探索和交易

### 2.3 视觉风格："希望朋克"

**不同于传统末日题材**:
- ❌ 绝望、压抑、黑暗（传统废土）
- ✅ 希望、重生、光明（希望朋克）

**视觉特征**:
- 废墟中的新生绿植
- 破败建筑上的彩色涂鸦
- 阳光穿透废墟的光影
- 温暖的色调（橙、绿、蓝）
- 手工感的UI元素（木板、手绘图标）

**情感基调**:
- 70% 温暖、希望、励志
- 20% 冒险、刺激、挑战
- 10% 紧张、危机（生存压力）

### 2.4 角色设定

**玩家角色**: 开拓者（Pioneer）
- 从避难所走出的幸存者
- 拥有GPS设备和基础生存工具
- 通过《新开拓者协议》获得土地开发权
- 背负着重建文明的使命

**NPC角色** (未来扩展):
- 协议广播员（教程引导）
- 商人（特殊交易）
- 探险家（任务发布）

## 3. 核心玩法机制

### 3.1 GPS圈地玩法

**设计理念**:
- 将"行走"转化为"占有"
- 现实世界的运动 = 游戏内的资产
- 类似小时候在沙滩上画圈圈

**操作流程**:
1. 点击"开始圈地"按钮
2. GPS开始记录轨迹
3. 屏幕显示蓝色轨迹线跟随玩家移动
4. 走回起点附近（< 20米）自动闭合
5. 系统验证合法性
6. 成功后变为绿色多边形，永久保存

**验证规则**:
| 规则 | 参数 | 理由 |
|------|------|------|
| 最小面积 | 100m² | 防止刷地，约等于一个小房间 |
| 最大面积 | 100,000m² | 防止霸占，约等于10个足球场 |
| 最小边长 | 5米 | 防止狭长条 |
| 最大速度 | 8km/h | 防止开车作弊 |
| GPS精度 | < 20米 | 信号不好禁止圈地 |
| 重叠检测 | 0容忍 | 不可与他人领地重叠 |

**防作弊机制**:
- GPS速度持续监测（不能开车/骑车）
- 轨迹平滑度检测（不能传送）
- 定位源验证（必须真实GPS）
- 时间合理性检测（面积与时间匹配）

**奖励设计**:
- 基础奖励: 金币 = 面积 × 0.1
- 首次圈地: 额外奖励200金币
- 完美闭合（误差<5米）: +50金币
- 大面积奖励（>50,000m²）: +500金币

### 3.2 探索搜刮玩法

**设计理念**:
- Walk to Earn - 走路就能赚资源
- 将真实地标转化为资源点
- 鼓励玩家探索周边环境

**POI类型与资源对应**:
| POI类型 | 代表地点 | 掉落资源 | 掉落量 | 稀有度 |
|---------|----------|----------|--------|--------|
| 医疗 | 医院、诊所、药店 | 急救包、抗生素、绷带 | 5-15个 | 稀有 |
| 食物 | 超市、餐厅、便利店 | 罐头、饮用水、干粮 | 10-30个 | 常见 |
| 工业 | 工厂、工地、仓库 | 金属、工具、零件 | 5-20个 | 普通 |
| 自然 | 公园、林地、湖泊 | 木材、植物纤维 | 10-25个 | 常见 |
| 金融 | 银行、珠宝店 | 稀有材料、电子元件 | 1-5个 | 传说 |

**触发机制**:
- 进入POI 100米范围内触发发现
- 弹窗提示"发现兴趣点"
- 点击"搜刮"按钮开始
- 显示探索动画（3秒）
- 展示获得物品清单

**冷却时间**:
- 同一POI: 24小时冷却
- 同类型POI: 无冷却（鼓励多走动）
- 全局探索: 免费用户每天10次，付费无限

**掉落算法**:
```
基础掉落数量 = POI等级 × 随机系数(0.8-1.2)
稀有物品概率 = 基础概率 × (1 + VIP加成)
掉落物品 = 根据POI类型从掉落表随机抽取
```

**探索成就记录**:
- 探索距离（GPS累计）
- 探索时长
- 搜刮次数
- POI发现数量

### 3.3 建造玩法

**设计理念**:
- 资源的消耗出口
- 从无到有的成就感
- 策略性资源分配

**建筑分类**:

**Tier 1 - 生存基础** (解锁等级: 1级):
| 建筑 | 消耗资源 | 功能 | 建造时间 |
|------|----------|------|----------|
| 篝火 | 木材×30, 石头×20 | 提供温暖，微弱照明 | 即时 |
| 简易庇护所 | 木材×50, 布料×20 | 休息恢复，存档点 | 5分钟 |
| 小木箱 | 木材×20 | +20背包容量 | 即时 |

**Tier 2 - 功能扩展** (解锁等级: 5级):
| 建筑 | 消耗资源 | 功能 | 建造时间 |
|------|----------|------|----------|
| 小仓库 | 木材×80, 金属×40 | +50背包容量 | 15分钟 |
| 工作台 | 木材×60, 金属×30 | 合成道具，修理工具 | 10分钟 |
| 水井 | 石头×100, 金属×50 | 每小时产出饮用水×2 | 30分钟 |
| 农田 | 木材×40, 种子×10 | 每8小时产出食物×5 | 20分钟 |

**Tier 3 - 高级设施** (解锁等级: 10级):
| 建筑 | 消耗资源 | 功能 | 建造时间 |
|------|----------|------|----------|
| 大仓库 | 金属×200, 混凝土×100 | +100背包容量 | 1小时 |
| 太阳能板 | 电子元件×20, 金属×100 | 提供能源，加速生产 | 1小时 |
| 风力发电机 | 电子元件×30, 金属×150 | 高级能源，覆盖整个领地 | 2小时 |
| 无线电塔 | 电子元件×50, 金属×200 | 通讯范围+50% | 2小时 |

**建造限制**:
- 只能在自己领地内建造
- 建筑数量上限 = 领地面积 ÷ 1000 (例: 10000m²领地最多10个建筑)
- 建造需要消耗资源（不可退回）
- 建造中状态可取消（返回50%资源）

**建筑状态机**:
```
建造中 (BUILDING)
    ↓ (时间到达)
待激活 (INACTIVE)
    ↓ (玩家激活)
运行中 (ACTIVE) ←→ 损坏 (DAMAGED)
    ↓ (玩家拆除)         ↓ (玩家维修)
已拆除 (DEMOLISHED)    运行中 (ACTIVE)
```

**建筑交互**:
- 点击建筑显示详情面板
- 可操作: 激活/停用、升级、维修、拆除
- 升级需求: 等级×2资源，+50%功能
- 拆除返还: 30%初始资源

### 3.4 交易玩法

**设计理念**:
- 资源流通促进社交
- 以物易物符合末日设定
- 基于位置增加真实感

**交易流程**:
1. **发布挂单**:
   - 选择"我提供"的物品（可多选）
   - 选择"我需要"的物品（可多选）
   - 设置交易比例（1:1或自定义）
   - 开启"允许交易"开关
   - 发布到附近

2. **发现挂单**:
   - 进入其他玩家领地100米范围
   - 自动显示该玩家发布的挂单
   - 浏览挂单列表

3. **接受交易**:
   - 点击感兴趣的挂单
   - 确认物品和比例
   - 系统检查双方资源是否充足
   - 点击"接受交易"
   - 原子性交换（要么成功，要么失败）

4. **完成交易**:
   - 双方背包自动扣除/增加物品
   - 交易记录保存
   - 双方获得经验值奖励

**交易限制**:
- 必须双方都开启"允许交易"
- 必须在100米范围内
- 每次交易最多10种物品
- 每天交易次数: 免费10次，VIP无限

**交易推荐算法**:
- 分析玩家缺少的资源
- 匹配附近有货的挂单
- 优先推荐"供需互补"的交易

### 3.5 生存体征玩法

**设计理念**:
- 持续的生存压力
- 资源的消耗循环
- 硬核玩家的挑战

**三大体征**:
| 体征 | 初始值 | 衰减速度 | 低于50%后果 | 低于20%后果 |
|------|--------|----------|-------------|-------------|
| 核心生命 | 100 | 不自动衰减 | - | 死亡重生 |
| 饱食度 | 100% | -5%/小时 | 探索距离-20% | 生命-1/小时 |
| 水分 | 100% | -10%/小时 | 建造速度-30% | 生命-2/小时 |

**恢复机制**:
| 物品 | 恢复效果 | 消耗 |
|------|----------|------|
| 罐头食品 | 饱食度+30% | 1个 |
| 干粮 | 饱食度+15% | 1个 |
| 饮用水 | 水分+40% | 1个 |
| 急救包 | 生命+50 | 1个 |

**增益机制**:
- 饱食度 > 80%: 探索资源掉落+20%
- 水分 > 80%: 建造速度+30%
- 双项 > 80%: 经验获取+50%

**死亡惩罚**:
- 生命归零触发死亡
- 损失当前背包10%物品（随机）
- 传送回主领地
- 所有体征恢复至50%
- 24小时内不可再次死亡（保护期）

## 4. 游戏系统设计

### 4.1 等级与经验系统

**经验来源**:
| 行为 | 经验值 | 每日上限 |
|------|--------|----------|
| 圈地 | 面积 ÷ 10 | 无上限 |
| 探索POI | 50 | 500 (10次) |
| 建造建筑 | 建筑等级 × 100 | 无上限 |
| 完成交易 | 30 | 300 (10次) |
| 完成成就 | 50-500 | 无上限 |

**等级设计**:
```
等级1-10: 新手期（快速升级，每级1000经验）
等级11-20: 成长期（中速升级，每级2000经验）
等级21-30: 进阶期（慢速升级，每级5000经验）
等级31+: 大师期（极慢升级，每级10000经验）
```

**等级奖励**:
- 每级: 金币×100
- 5级: 解锁Tier 2建筑
- 10级: 解锁Tier 3建筑
- 15级: 解锁卫星通讯
- 20级: 解锁高级交易
- 30级: 解锁传说成就

### 4.2 货币系统

**金币（主货币）**:
- 获取途径:
  - 圈地奖励
  - 等级提升
  - 成就奖励
  - 每日任务
- 用途:
  - 加速建造
  - 购买建筑图纸
  - 背包扩容
  - 领地维护费

**钻石（付费货币）**:
- 获取途径:
  - 内购充值
  - 稀有成就
- 用途:
  - 购买VIP
  - 购买稀有资源包
  - 复活（避免死亡惩罚）

**汇率设计**:
- 100钻石 = $0.99
- 金币不可直接兑换钻石
- 部分内容可用金币或钻石

### 4.3 成就系统

**成就分类**:

**新手成就** (白色):
- 第一次圈地: 100金币
- 第一次探索: 100金币
- 第一次建造: 100金币
- 第一次交易: 100金币
- 加入频道: 50金币

**进阶成就** (蓝色):
- 圈地10次: 500金币
- 探索50个POI: 500金币
- 建造20个建筑: 500金币
- 交易100次: 1000金币

**稀有成就** (紫色):
- 领地总面积>100,000m²: 2000金币
- 探索所有类型POI: 2000金币
- 收集所有Tier 2建筑: 2000金币
- 登上排行榜前10: 3000金币

**传说成就** (橙色):
- "领地之王": 拥有10块领地: 5000金币 + 专属称号
- "废墟探险家": 探索500个POI: 5000金币 + 专属头像框
- "建造大师": 建造100个建筑: 5000金币 + 专属建筑皮肤
- "传奇交易者": 完成1000次交易: 10000金币 + 交易手续费永久免除

### 4.4 排行榜系统

**三大榜单**:
1. **领地面积榜**:
   - 统计玩家所有领地总面积
   - 前10名每周奖励: 1000金币
   - 前100名显示

2. **探索废墟榜**:
   - 统计探索POI数量
   - 前10名每周奖励: 800金币
   - 鼓励持续探索

3. **建筑数量榜**:
   - 统计所有建筑数量
   - 前10名每周奖励: 600金币
   - 鼓励建设投入

**排行榜规则**:
- 每5分钟更新一次（缓存）
- 只显示活跃玩家（7天内登录）
- 可按地区/全球筛选
- 点击玩家查看其领地（地图模式）

## 5. 数值平衡设计

### 5.1 资源产出与消耗平衡

**每日资源产出**（活跃玩家）:
- 探索10次POI: 约150个基础资源
- 农田/水井产出: 约30个生存资源
- 交易获得: 约50个目标资源

**每日资源消耗**:
- 生存消耗（饱食度/水分）: 约40个生存资源
- 建造新建筑（1-2个）: 约100个建筑资源
- 建筑维护: 约20个维护资源

**平衡点**: 产出 > 消耗 20-30%，保证玩家有成长感

### 5.2 建造时间设计

**建造时间梯度**:
- Tier 1建筑: 即时-5分钟（让新手快速体验）
- Tier 2建筑: 10-30分钟（中期挑战）
- Tier 3建筑: 1-2小时（后期投入）

**加速设计**:
- 金币加速: 1金币 = 1分钟
- 钻石加速: 1钻石 = 10分钟
- VIP加速: 建造时间自动减半

### 5.3 探索频率设计

**免费玩家**:
- 每天10次探索机会
- 24小时恢复
- 鼓励每天登录

**VIP玩家**:
- 无限探索
- 探索范围+50%（100米→150米）
- 掉落物品+20%

**平衡考虑**:
- 免费玩家每天10次足够完成日常
- VIP提供便利而非碾压优势
- 付费/免费差距控制在2倍以内

### 5.4 交易手续费设计

**手续费机制**:

**防刷机制**:
- 同一对玩家每天交易上限: 10次
- 新玩家（<3级）不可交易
- 异常交易（价值悬殊）人工审核

### 10.1 导航架构

**底部Tab导航**:
1. **地图** (默认页):
   - 卫星地图
   - 圈地按钮
   - 我的位置按钮

2. **资源**:
   - 背包（物品列表）
   - 建造（建筑列表）
   - 交易（挂单列表）

3. **通讯**:
   - 消息（官方推送）
   - 公共频道
   - 私人频道
   - PTT呼叫
   - 设备升级

4. **个人**:
   - 玩家档案
   - 领地列表
   - 排行榜
   - 成就
   - 设置

### 10.2 关键交互设计

**圈地操作**:
1. 点击"开始圈地"大按钮
2. 按钮变为"圈地中..."（脉冲动画）
3. 地图上实时绘制蓝色轨迹
4. 右上角显示:
   - 当前距离
   - 预估面积
   - GPS精度
5. 接近起点时显示"即将闭合"提示
6. 闭合成功:
   - 轨迹变绿色
   - 播放成功音效
   - 弹出奖励面板（金币+成就）

**探索操作**:
1. 靠近POI自动弹出卡片（从底部滑入）
2. 卡片显示:
   - POI名称和类型
   - 预计掉落资源图标
   - 冷却时间（如有）
3. 点击"搜刮"按钮
4. 播放3秒搜刮动画（翻箱倒柜）
5. 弹出战利品列表（卡牌翻转效果）
6. 滑动关闭，物品自动进入背包

**建造操作**:
1. 点击"建造"按钮
2. 弹出建筑分类筛选
3. 选择建筑，显示详情页:
   - 建筑图标（3D视图）
   - 资源需求清单（缺少的标红）
   - 功能说明
   - "开始建造"按钮
4. 点击建造后进入放置模式:
   - 建筑图标跟随手指移动
   - 显示半透明预览
   - 有效位置绿色框，无效位置红色框
5. 松手放置，扣除资源，开始建造
6. 地图上显示建造中状态（进度条）

### 10.3 反馈机制设计

**即时反馈**:
- 任何操作都有音效
- 任何状态变化都有动画
- 任何奖励都有粒子特效

**进度反馈**:
- 建造中: 进度条 + 剩余时间
- 体征衰减: 血条颜色变化（绿→黄→红）
- 等级经验: 顶部经验条实时更新

**错误反馈**:
- 资源不足: 红色抖动 + "资源不足"提示
- GPS信号差: 黄色警告 + "等待GPS信号"
- 操作冷却: "XX后可再次操作"

### 10.4 无障碍设计

**字体**:
- 最小字号: 14pt（符合iOS规范）
- 重要信息: 加粗或放大
- 支持系统字体大小调节

**色盲友好**:
- 不仅依赖颜色区分状态
- 配合图标、文字说明

**单手操作**:
- 常用按钮在屏幕下半部分
- 底部Tab高度60pt（易点击）

## 11. 技术实现要点

### 11.1 性能优化

**地图优化**:
- 领地多边形超过50个时分批渲染
- 使用MapKit的Cluster（聚合）显示大量POI
- 缩放级别低时隐藏建筑图标

**数据同步**:
- 本地优先，异步同步云端
- 增量同步（只传输变化部分）
- 冲突解决策略（服务器优先）

**内存管理**:
- 图片使用WebP格式（体积小50%）
- 离开页面释放大图
- 使用对象池管理频繁创建的对象

### 11.2 安全设计

**防作弊**:
- 关键计算服务器验证（圈地面积、资源掉落）
- GPS轨迹完整上传服务器检测
- 异常行为封号机制

**数据安全**:
- 用户密码BCrypt加密存储
- API通信HTTPS加密
- 敏感数据（VIP状态）服务器双重验证
